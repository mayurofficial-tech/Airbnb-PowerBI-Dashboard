# 📌 Project Overview

This project analyzes **Airbnb performance across major global cities** using Power BI.  
The dashboard provides insights into listings, hosts, reviews, pricing, ratings, and seasonal trends.

The goal of this project is to demonstrate **data visualization, business analysis, and storytelling using Power BI**.


---

# 📈 Key Metrics

- **279,712 Total Listings**
- **182,024 Hosts**
- **5,372,983 Reviews**
- **144 Property Types**
- **10 Major Cities Analyzed**

---

# 🔍 Key Insights

• **Paris, New York, and Sydney** account for nearly **50% of total listings**.

• **Hotel rooms have the highest average price** compared to other property types.

• **Mexico City and Rio de Janeiro** have the highest overall ratings.

• **Hong Kong and Istanbul** show relatively lower average ratings.

• **98.8% of customers leave reviews three times or fewer**, indicating low repeat review frequency.

• **European cities show strong seasonality**, with peak review activity from **April to August**.

• **Over two-thirds of Airbnb hosts are identity verified**, increasing platform trust.
